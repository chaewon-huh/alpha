# Alpha Timer

Minimal timer for macOS with liquid glass effect.

![Alpha Timer Screenshot](./assets/alpha-timer-preview.png)

## Requirements

- macOS 13+
- Node.js 22+

## Install

```bash
npm install
npm start
```

## Features

- Liquid glass UI (electron-liquid-glass)
- Multiple timers
- SF time display
- Keyboard shortcuts (Cmd+N, Cmd+S, Esc)
- Draggable windows
- Font scaling (Cmd +/-)

## Build

```bash
npm run dist
```
